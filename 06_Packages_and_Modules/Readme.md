# Packages

GOROOT -> installation directory
GOPATH -> Workspace Directory

Recommended Folder Structure

GOROOT
    * src -> source file for standard library
    * pkg -> complied pasckages
    * bin -> executable commanda
GOPATH
    * src -> code
    * pkg -> thirdparty libraries
    * bin -> executables

if multiple init functions it will be executed in the alphabetical order of the file names
